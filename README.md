# Website UI Redesign
## 📌 Project Overview
This project focuses on redesigning an existing website to improve usability and user experience using UI/UX principles.
## 🎯 Objectives
- Identify usability issues using heuristic evaluation
- Improve navigation and layout
- Create a clean and modern UI
- Ensure responsive design
## ⚠️ Problems Identified
- Poor navigation
- Cluttered interface
- Lack of user feedback
- Inconsistent design
## 👤 User Persona
**Name:** Priya  
**Goal:** Quick and easy food ordering  
**Pain Points:** Confusing layout and slow process  
## 🎨 Design Process
1. Heuristic Evaluation  
2. Mood Board Creation  
3. Style Guide Design  
4. Wireframing  
5. High-Fidelity UI Design  
6. Responsive Design  
## 💻 Tools Used
- Figma
## 📱 Features
- Clean and minimal UI  
- Easy navigation  
- Responsive design for mobile and desktop
